## ðŸš€ Quick Start Guide

### 1. Prerequisites
* **Install MoreLogin:** Download and install the client from the [Official Download Page](https://www.morelogin.com/download/).
* **Activate Local API:** * Open the MoreLogin Client.
    * Navigate to **Settings** -> **Local API**.
    * Ensure the status shows **"Success"** (Default Port: `40000`).

### 2. Installation
Clone this repository to your local machine:
```bash
git clone [https://github.com/MoreLoginBrowser/MoreLogin-API-Demos.git](https://github.com/MoreLoginBrowser/MoreLogin-API-Demos.git)
cd MoreLogin-API-Demos

```

### 3. Run a Demo (Python Example)

```bash
cd MoreLogin-Python
pip install -r requirements.txt
python your_script_name.py

```

## ðŸ“– Key API Capabilities

Our [Local API Reference](https://guide.morelogin.com/api-reference/local-api) covers:

* **Profile Management:** `POST http://127.0.0.1:40000/api/env/page`, `POST http://127.0.0.1:40000/api/env/create/quick`

* **Proxy Integration:** Dynamically bind SOCKS5, HTTP, or SSH proxies to profiles.
* **Cloud Phone Management:** [Open API](https://guide.morelogin.com/api-reference/open-api).
